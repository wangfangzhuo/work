## 变更说明（v3 → v4）

| 维度 | v3 | v4（本版） |
|------|-----|-----------|
| 搜索入口页 | 独立提示词A，简单悬浮圆球 | **完整1药城搜索页**，含导航栏/筛选器/商品网格/悬浮引导卡片 |
| 入口引导 | 展开为大卡片+柱状图 | **紧凑卡片**（右下角320px），含图片+文案+两个按钮 |
| 页面切换 | 两个独立HTML | **单页面双视图**，搜索页↔对话界面可互相切换 |
| 初始平台状态 | 全部未连接 | **4个线上平台默认已连接已授权**，九州通未连接 |
| 异常处理 | 采集后直接展示异常行 | **新增 AuthDecision 卡片**：授权过期时先让用户选择"立即配置"或"跳过" |
| 采集状态 | 只有 done/pending/loading | 增加 **expired（授权过期）** 和 **error（获取失败）** 状态 |
| 配置后恢复 | 无 | **pendingSearch**：用户选择配置后，配置完成自动恢复比价流程 |
| 关键词异常 | 阿莫西林→小药药过期+九州通无此药 | 阿莫西林→**小药药过期**；感康→**药师帮获取失败** |

## 关键词触发规则汇总

| 输入关键词 | 触发场景 | 异常平台 | 特殊交互 |
|-----------|---------|---------|---------|
| 感冒灵（或含「感冒」） | 正常场景 | 无 | 4平台全部正常 |
| 阿莫西林（或含「莫西」） | 授权过期场景 | 小药药 expired | AuthDecision 二选一 + pendingSearch 恢复 |
| 感康 | 数据获取失败场景 | 药师帮 error | hover tooltip 提示原因，直接跳过 |
| 结算/购物车/汇总 | 采购计划场景 | — | 清单汇总 + 结算 |
| 其他任意输入 | 走感冒灵正常流程 | 无 | 用输入的词替换药品名 |
