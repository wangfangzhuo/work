# 鼎新 Demo 提示词 v3

> 记录时间：2026-03-16
> 基于：方拙代码交互（ai采购---对话式药店采购助手）完全重写
> 用途：发给 Google AI Studio 生成可交互 HTML Demo
> 单页面：纯对话界面（无搜索入口页、无侧边栏）

---

## 变更说明

相比 v2（融合版）的主要变化：

| 维度 | v2 | v3（本版） |
|------|-----|-----------|
| 设计基准 | 淼鑫文档 + 代码融合 | **完全基于代码**，去掉所有淼鑫设计 |
| 页面结构 | 左侧边栏 + 右侧对话 | **单屏对话**，无侧边栏、无历史对话 |
| 搜索入口页 | 有 | **无**，直接进入对话 |
| 平台数量 | 8个（含A/B/C/D） | **5个**：1药城、药师帮、小药药、药九九、九州通（线下） |
| 流程步骤 | 消歧 → 规格确认 → 查价 | **消歧 → 采集 → 规格确认 → 比价**（与代码一致，采集在规格确认前） |
| 结算按钮 | 单按钮"确认并去结算" | **两个按钮**：线上一键加车 + 下载采购计划 |
| 悬浮球 | 有 | **无** |

```

【产品背景】
鼎新是嵌在1药城的 AI 药品采购决策助手。
用户输入药品名，AI 先做药品消歧（让用户确认具体品种），
```
