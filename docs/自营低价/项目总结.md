# 自营低价项目总结

## 一、项目定位

1药城平台通过"竞价直通车（JBP）"机制，驱动商家主动卷向更低价格，形成 **低价 → 流量增长 → 交易增长** 的正向循环，提升平台整体价格竞争力和动销水平。

## 二、核心机制：竞价直通车

### 规则概述
- **活动时间：** 长期有效
- **竞价规则：** 商家将商品价格调至1药城自营最低价，系统自动竞价、实时生效（最迟20分钟）
- **商家权益：**
  - 搜索置顶曝光（首位展示）
  - 智能采购曝光（匹配结果前三位）
  - 打标"自营低价"
  - 成本可控，无需额外费用投入

### 圈品条件（保障客户体验）
- 库存 ≥ 100
- 效期 ≥ 1年
- 限购：全国连锁周限购 ≥ 200
- 控销：全国 ≥ 20个省市区可售

### 竞价优先级
- 推荐竞价商品：当前价格排位第2-3名，距离第一名仅一步之遥的品种
- 价格相同时，优选库存较多的商家

### 剔除机制
库存 < 100、效期 < 1年、限购/控销不达标的商品自动掉标

## 三、当前数据表现

### 核心指标
| 指标 | 目标 | 现状 |
|------|------|------|
| 参与竞价商家覆盖占比 | 95% | 稳定达成 |
| 周竞价次数 | 50W+ | W10约45.8W次，有所下滑 |
| 低价订单GMV占比 | 65% | 约50%，未达预期 |

### 关键问题
- **份额错位：** T700核心品中58%主码已打标，但打标SKU仅贡献27%的GMV
- **虹吸失灵：** >70%的订单流向了非低价SKU
- **活跃衰减：** 3月取消考核后，竞价次数从最高每周50W+降至40W+

## 四、问题诊断：三大"断裂"

### 1. 价格效能断裂（最便宜的卖不动）
- **属地化瓶颈：** 异地低价受限于运费和时效，客户倾向就近仓下单
- **价格优势不明显：** 部分低价品仅是"数字最低"，缺乏"体感差"的绝对价格优势
- **非价格因素影响：** 客户要求整件不散货、效期偏好、库存数量要求等

### 2. 商家动力断裂（商家不愿卷）
- **获益感知弱：** 低价份额仅27%，商家付出利润未换回规模
- **信息触达窄：** 仅触达2-3名商家，大量潜在商家处于围观状态
- **运营成本高：** 缺乏自动化工具，人工逐条确认调价效率低

### 3. 打标准确性问题（86%的TOP未打标问题）
- 针对客户类型限销但对连锁可售（78%）
- 有周限购设置但均大于200盒（22%）
- 前台显示有货但可售库存不足（20%）
- 近效期品种也打标（2%）

## 五、行动策略

### 运营侧
1. **过程管理：** 落后人员每周复盘，推动定向调价
2. **N700品种跟进：** 按天结合中标价跟进到采购/运营，生成定向调价清单
3. **Open Claw接入准备：** 业务经验结构化 → RAG知识库 → AI定价Skill（跟价、时段调价、毛利风控、冲销）

### 产品侧
| 方向 | 具体措施 | 进展 |
|------|---------|------|
| 自营竞价效率 | PIS兜底+自动跟价提升N700竞价次数和成功率 | 待开发 |
| 价格力提升 | PIS价格加入低价比价 | 待开发 |
| 低价曝光提升 | 详情页引导、智采强化低价引导、下单新增低价引导 | 3月确认方案 |
| 买家信任 | 按属地比价，监控低价准确性 | 规划中 |
| 商家竞价效率 | 自动跟价、低价品补货/效期预警、销售增长信息展示 | 规划中 |

### 考核指标重构
- **从"卷动作"到"卷结果"：** 由"调价次数"转向"自营低价品GMV贡献占比"
- **分档评估：** 按主码维度统计低价GMV占比，>80%充分竞价，40-80%中等，<40%需提升
- **目标：** 核心品种低价成交份额由27%提升至50%以上
