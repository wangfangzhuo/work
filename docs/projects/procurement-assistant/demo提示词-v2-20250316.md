# 鼎新 Demo 提示词 v2（融合版）

> 记录时间：2026-03-16
> 基于：淼鑫代码交互 + 方拙场景文档 + 产品形态设计 融合重写
> 用途：发给 Google AI Studio 生成可交互 HTML Demo
> 演示顺序：00-search-entry → 03-auth → 04-chat

---

## 变更说明

相比 v1（2026-03-15）的主要变化：

| 维度 | v1 | v2（本版） |
|------|-----|-----------|
| 搜索入口页 | 静态弹窗 | 增加已授权状态下的"价格预览+快速比价" |
| 对话界面 | 纯文档描述 | 融合淼鑫代码的 UI 组件（供应商管理面板、规格确认卡片、比价表格、购物车） |
| 供应商管理 | 仅侧边栏列表 | 内联在对话流中，支持在线平台授权 + 线下报价单上传 |
| 规格确认 | 仅药品消歧 | 两步确认：先消歧药品品种 → 再推荐具体规格+厂家（用户可更换，带历史采购/热门/性价比标签） |
| 比价结果 | 纯列表 | 结构化表格（价格/活动/效期/服务评分/物流），带一键采纳最优方案 |
| 异常处理 | 有 | 保留，增加内联重新授权表单 |
| 购物车 | 无 | 增加购物车汇总 + 采购计划下载（线上辅助加车+下载计划/线下下载采购计划） |
| 报价单上传 | 无 | 支持拖拽/点击上传，带解析进度动画 |
