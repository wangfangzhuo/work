## 二、地址三字段的提取来源规则

注册地址、销售单打印地址、收货地址，不同企业类型从不同证照提取：

| 企业类型 | 地址提取来源 |
|---------|-----------|
| 单体药店、连锁总店、批发企业、连锁特许经营 | **药品经营许可证** |
| 非公立医疗机构-营利性、个体诊所-营利性、生产企业 | **营业执照** |
| 非公立医疗机构-非营利性、个体诊所-非营利性、公立医疗 | **医疗机构执业许可证/备案凭证** |

## 五、各证照审核标准

### 营业执照

| 审核项 | 要求 |
|-------|------|
| 公示系统核验 | 国家企业信用信息公示系统，只看是否注销，其他不看 |
| 企业名称/法人/地址 | 须与公示系统一致 |
| 变更处理 | 名称/法人/地址变更→退回，原因："企业名称或法定代表人、地址变更为XXX，请更新整套资质" |
| 注销处理 | 直接退回，原因："统一社会信用代码XXX已注销，不可采购" |
| 经营范围勾选 | 化妆品、消毒制品、普通商品、一类器械→全勾选（无论执照有没有）；其余有才勾 |

### 药品经营许可证

| 审核项 | 要求 |
|-------|------|
| 公示系统核验 | 国家药品监督管理局，只看批发企业，只看是否注销 |
| 企业类型判断 | 编号首字母：A→批发、B→连锁总店、C→连锁特许经营、D→单体药店（配合名称及经营方式） |
| 经营范围勾选 | 严格按许可证经营范围勾选，客户多选则取消，少选则添加 |
| 未更新资质 | 不可修改经营范围，系统会拦截 |

### 医疗机构执业许可证/诊所备案凭证

| 审核项 | 要求 |
|-------|------|
| 企业类型判断 | 经营性质"政府办"→公立医疗机构 |
| 名称不一致 | 营业执照"XX诊所有限公司" vs 许可证"XX诊所"，若地址、法人一致→可通过 |
| 经营范围 | 根据诊疗科目判断：含中医科/全科/中西医结合科→可勾中药；不含→不可勾中药材/中药饮片 |

### 采购/收货委托书

| 审核项 | 要求 |
|-------|------|
| 采购人 | 填姓名+身份证号，须与身份证框的人同一人 |
| 收货人 | 填姓名+身份证号 |
| 收货员签名 | 至少一人与上面收货人同一人，须手写或盖章 |
| 委托期限 | 必须填写 |
| 法定代表人 | 签名或盖法人章，或企业（主要）负责人签字/盖章 |
| 公章 | 盖该店公章 |
| 签发日期 | 在生效日期当天或之前 |
| 身份证贴附 | 身份证若是贴上的需盖店公章在身份证上 |

### 通用图片要求

| 要求 | 说明 |
|------|------|
| 清晰完整 | 无其他物品、无背景图片、无边角遮挡 |
| 公章真实 | 加盖购货单位公章原印章，不接受抠章 |
| 不接受抠图 | 身份证不接受抠图 |
| 公章与证件同页 | 公章与身份证必须在同一张纸上 |

## 六、特殊情况处理

| 场景 | 处理方式 |
|------|--------|
| 证件间地址不一致无法判断同一地址 | 需居委会/街道办/村委/派出所盖章证明 |
| 经营地址搬迁 | 需药监部门变更记录或新证件，不可凭地址证明 |
| 各证照地址详细程度不同（如一个到村号一个到村） | 无变更情况→可不提供证明 |
| 不同信用代码的企业更换证照 | 不可在原账号修改，须注销后重新注册 |
| 广东福建等地药品食品器械在一个证上 | 同一个证分别传到三个资质框 |
| 连锁特许经营 | 有的子公司不可对连锁门店供货；系统已自动设置审核 |
| 食品备案和食品许可证都上传 | 填发证日期最新的编号和有效期 |
| 证上没有有效期 | 系统默认2099-12-31 |
| 连锁总店/批发企业药证有多个仓库地址 | 填一个即可，可后台添加多地址白名单 |
| 药证仓库地址写"由XX公司配送" | 需附配送公司盖章药证，收货地址按配送公司仓库地址 |
